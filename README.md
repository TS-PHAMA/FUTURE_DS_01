# FUTURE_DS_01

Business Sales Performance Analytics

Future Interns – Data Science & Analytics Task 1

## Project Overview

This project analyzes retail sales data to identify revenue trends,
top-performing products, profitable categories, and regional performance.

The objective is to generate actionable business insights that can support
decision-making and improve profitability.

## Business Questions

1. Which products generate the highest revenue?
2. Which products generate the highest profit?
3. How do sales change over time?
4. Which categories perform best?
5. Which regions contribute most to profit?
6. What business recommendations can improve performance?

## Dataset
Source: Sample Superstore Dataset
Observations: 9,994

Variables:
- Sales
- Profit
- Quantity
- Category
- Sub-Category
- Region
- State
- Product Name
- Order Date
